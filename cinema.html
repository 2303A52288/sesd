Friday: Movie Ticket Booking System

A cinema chain is developing an online ticket booking system that allows users to select movies, showtimes, and book seats. The system must ensure that no seat is double-booked and provide the option for users to cancel their bookings with refunds.

Task Requirements

  • Design the following classes:
    • Movie
    • Theater
    • Ticket
    • Booking
  • Implement core features:
    • Seat booking functionality
    • Seat availability check
    • Booking cancellation and refund
  • Write unit tests to verify:
    • Seat availability updates correctly after booking.
    • Double booking of the same seat is prevented.
    • Cancellation restores seat availability and processes refund correctly.
